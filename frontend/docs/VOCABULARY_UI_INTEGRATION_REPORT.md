## 📚 사용자 여정 검증

### 1. 신규 사용자 여정
1. ✅ 메인 페이지 → 단어장 링크 클릭
2. ✅ 빈 단어장 상태 → 검색으로 단어 추가 유도
3. ✅ 단어 검색 → 관련 단어 발견 → 단어장 추가
4. ✅ 복습 시작 → 기본 설정으로 플래시카드 모드

### 2. 기존 사용자 여정  
1. ✅ 단어장 진입 → 기존 단어들 표시
2. ✅ 필터링/정렬 → 원하는 단어 찾기
3. ✅ 단어 상세 → 편집/삭제/복습
4. ✅ 복습 모드 선택 → 진행도 표시 → 완료

### 3. 복습 여정
1. ✅ 복습 대상 단어 자동 선별
2. ✅ 모드 선택 (플래시카드/빈칸채우기/철자게임)
3. ✅ 실시간 진행도 표시
4. ✅ 결과 요약 → 틀린 단어 재복습 옵션
